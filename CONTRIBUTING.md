# Contributing to MediGuide AI

Thank you for your interest in contributing to MediGuide AI!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/MediGuide_AI.git`
3. Follow the setup instructions in [SETUP.md](SETUP.md)
4. Create a new branch: `git checkout -b feature/your-feature-name`
5. Make your changes
6. Test your changes: `npm run dev`
7. Commit your changes: `git commit -m "Add your feature"`
8. Push to your fork: `git push origin feature/your-feature-name`
9. Create a Pull Request

## Code Style

- Use functional components with hooks
- Follow React best practices
- Use Material-UI components for consistency
- Write meaningful commit messages
- Comment complex logic

## Testing

Before submitting a PR:
- Ensure the app builds: `npm run build`
- Test all features manually
- Check responsive design on different screen sizes
